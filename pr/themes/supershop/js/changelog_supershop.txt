==> So, how to upgrade the "Group Category" module from version 1.0 to version 2.0

***********     IMPORTANT Notes    ***********

- Before you upgrade or change anything,... you need to backup both of the database and source code of your site.

- The module was rebuilt, I made a function that allows it to convert the data/information of the old version to a new version of this module. 

- If you setup the Supershop theme from version 2.7 then you do not need to upgrade this module.

- If there is any issues when you upgrade, pls send us an email : kutethemes@gmail.com (in the email, pls embed the purchase code. Thanks!)

**********************************************

To update the module, you can do following these steps: 

+ Step 1. You need to download the upgarded-package for this module here: http://kutethemes.com/demo/supershop/upgrade/upgrade.zip. Then Unzip the upgrade.zip file. This file contains the folders and new files as well as the updated files.

+ Step 2. Copy the modules and themes folders into the root folder of your website. So the changed/updated files will be overwritten,...and new files will be copied into the correct positions,... so you do not have to update file by file manually. 

+ Step 3. Login with the backend then click on the "modules" link in the left menu of backend. Prestashop will update this module automatically to version 2.0.

+ Step 4. Clear cache of your site (in the backend).
